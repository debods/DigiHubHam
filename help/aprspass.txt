APRS Password Generator (aprspass)
===================================

NAME
    aprspass - Generate the APRS-IS passcode for a US amateur radio
    callsign.

SYNOPSIS
    aprspass <callsign>

DESCRIPTION
    aprspass validates the given callsign with validcall.py, then
    computes its APRS-IS passcode with aprspass.py (the standard
    APRS-IS passcode algorithm, keyed on the callsign only; any SSID
    or portable/mobile suffix is ignored).

    The passcode is deterministic: the same callsign always produces
    the same password. DigiHub stores this value in field 17 of
    $HOME/.dhinfo and keeps it in sync automatically via dhedit
    whenever the installed callsign changes.

ARGUMENTS
    callsign
        The amateur radio callsign to generate a passcode for.
        Required.

EXAMPLES
    Generate the APRS passcode for W0FFS:
        aprspass W0FFS

FILES
    /usr/local/bin/validcall.py
        Validates the callsign format before generating a passcode.

    /usr/local/bin/aprspass.py
        Computes the passcode itself.

EXIT STATUS
    0
        Passcode generated and printed successfully.

    1
        The callsign failed validation, or an unexpected error
        occurred.

    2
        Incorrect syntax (a callsign argument was not supplied).
