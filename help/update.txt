System Package Update (update)
=================================

NAME
    update - Update and upgrade the operating system's packages.

SYNOPSIS
    update

DESCRIPTION
    update is a thin convenience wrapper that runs, in order:

        sudo apt update
        sudo apt -y full-upgrade
        sudo apt -y autoremove

    This updates the OS-level package set (kernel, system libraries,
    etc.), not DigiHub's own scripts. To check for and apply updates
    to DigiHub itself, use "dhupdate" instead.

EXAMPLES
    Update and upgrade all system packages:
        update

EXIT STATUS
    Reflects the exit status of the last apt command run. There is no
    error handling between steps; if "apt update" fails, the
    following upgrade/autoremove steps still run against whatever
    package index is currently available.
