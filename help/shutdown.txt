System Shutdown (shutdown)
============================

NAME
    shutdown - Power off the system.

SYNOPSIS
    shutdown

DESCRIPTION
    shutdown is a thin convenience wrapper around
    "sudo systemctl poweroff". It takes no arguments and prompts for
    a sudo password if the user's cached sudo credentials have
    expired.

EXAMPLES
    Power off the system:
        shutdown

EXIT STATUS
    Reflects the exit status of "sudo systemctl poweroff". On success
    the system powers off and this command does not return.
