System Reboot (reboot)
========================

NAME
    reboot - Reboot the system.

SYNOPSIS
    reboot

DESCRIPTION
    reboot is a thin convenience wrapper around
    "sudo systemctl reboot". It takes no arguments and prompts for a
    sudo password if the user's cached sudo credentials have expired.

EXAMPLES
    Reboot the system:
        reboot

EXIT STATUS
    Reflects the exit status of "sudo systemctl reboot". On success
    the system reboots and this command does not return.
