Current GPS Position (position)
=================================

NAME
    position - Read and display the current position from the
    configured GPS device.

SYNOPSIS
    position

DESCRIPTION
    position activates the DigiHub Python virtual environment, calls
    position.py to read a fix from the configured GPS device, then
    calculates the Maidenhead grid square for that fix with
    maidenhead.py.

    This is a one-shot, interactive read: it waits up to
    position.py's default timeout (5 seconds) for a fix and prints
    whatever it receives. It does not update $HOME/.dhinfo; that is
    handled continuously in the background by the dhgpsmonitor
    service when a GPS device is configured.

    If no GPS device is configured (DigiHubGPSport is "nogps"),
    position prints an error and exits without attempting a read. If
    a device is configured but does not return a fix in time, no
    coordinates are printed.

EXAMPLES
    Read the current GPS position:
        position

FILES
    /usr/local/bin/position.py
        Reads the raw NMEA fix from the configured serial device.

    /usr/local/bin/maidenhead.py
        Calculates the grid square from the fix.

EXIT STATUS
    0
        Ran to completion (a fix may or may not have been printed,
        depending on whether the GPS device responded in time).

    1
        No GPS device is configured for DigiHub (DigiHubGPSport is
        "nogps"). Run dhedit and use "Manage GPS device" to add one.
