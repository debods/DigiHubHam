Maidenhead Grid Calculator (maidenhead)
=========================================

NAME
    maidenhead - Calculate a Maidenhead grid square from latitude and
    longitude.

SYNOPSIS
    maidenhead <latitude> <longitude>

DESCRIPTION
    maidenhead validates the given coordinates and computes the
    six-character Maidenhead grid square with maidenhead.py.

    This is the same calculation DigiHub uses internally whenever
    latitude or longitude changes (during installation, in dhedit,
    and in dhgpsmonitor).

ARGUMENTS
    latitude
        Decimal degrees, -90 to 90. Required.

    longitude
        Decimal degrees, -180 to 180. Required.

EXAMPLES
    Calculate the grid square for a coordinate:
        maidenhead 28.1899 -82.50036

FILES
    /usr/local/bin/maidenhead.py
        Performs the coordinate validation and grid calculation.

EXIT STATUS
    0
        Grid square calculated and printed successfully.

    1
        The supplied coordinates are out of range.

    2
        Incorrect syntax (latitude and/or longitude were not
        supplied, or could not be parsed as numbers).
