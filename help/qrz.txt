Callsign Lookup (qrz)
=======================

NAME
    qrz - Look up a US amateur radio callsign's FCC license and
    address information.

SYNOPSIS
    qrz <callsign>

DESCRIPTION
    qrz looks up a callsign using, in order:

      1. The local hamdb MariaDB database (populated by "hamdb full"
         at install time and refreshed daily by the dhhamdbupdate
         timer), enriched with grid/latitude/longitude from the
         hamdb.org API when available.
      2. If the local database is unavailable or has no record, the
         hamdb.org API directly.

    If neither source provides usable coordinates, and a Mapbox API
    token is configured (field 19 of $HOME/.dhinfo, set during
    installation or later via dhedit), qrz geocodes the license
    address as a fallback using geocode.py. This step is entirely
    optional: with no token configured, qrz simply skips it and
    shows whatever coordinates the other sources provided (if any).

    Results are rendered as a formatted operator/license/address/grid
    summary, noting which source(s) the data came from.

ARGUMENTS
    callsign
        The US amateur radio callsign to look up. Required. Must
        contain only letters, digits, and "/".

EXAMPLES
    Look up a callsign:
        qrz W0FFS

FILES
    $HOME/.hamdb.cnf
        MySQL/MariaDB credentials used for the local database lookup;
        see the hamdb help file.

    $HOME/.dhinfo
        Field 19 (mapboxtoken) is read for the optional geocoding
        fallback.

    /usr/local/bin/geocode.py
        Performs the optional Mapbox geocoding fallback.

EXIT STATUS
    0
        Callsign found and displayed successfully.

    1
        Incorrect syntax (no callsign supplied), or the callsign
        contains characters outside A-Z, 0-9, and "/".

    2
        Neither the local database nor the hamdb.org API returned a
        result.

    4
        A result was returned but did not match the requested
        callsign, or the callsign is not a valid/found US license.
