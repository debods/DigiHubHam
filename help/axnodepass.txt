AX Node Password Generator (axnodepass)
=========================================

NAME
    axnodepass - Generate a random alphanumeric AX.25 node password.

SYNOPSIS
    axnodepass

DESCRIPTION
    axnodepass generates a fresh 6-character random alphanumeric
    password, suitable for use as an AX.25 node login password.

    Unlike the APRS passcode, this password has no relationship to
    the callsign; it is purely random, generated with
    "openssl rand -base64 12 | tr -dc A-Za-z0-9 | head -c6" each time
    it runs. DigiHub stores the value used at install time in field
    18 of $HOME/.dhinfo; run dhedit to view, regenerate, or manually
    change it.

    Running axnodepass directly generates and displays a new
    password but does not save it anywhere; use dhedit to persist a
    new value into $HOME/.dhinfo.

EXAMPLES
    Generate a random AX node password:
        axnodepass

EXIT STATUS
    0
        Password generated and printed successfully.

    1
        The shared UI library could not be loaded.
